# Portfolio Snippet

Title: Doppel

Tagline: Synthetic user runtime for experiential product testing.

Short description: Doppel runs AI-driven synthetic users through real Web product flows, records session artifacts, extracts evidence, and generates structured UX evaluation reports. v0.1.0 focuses on Web page experience testing with browser-use runtime, persona configuration, judge skills, multi-persona batch runs, runtime fallback and Markdown/JSON reports.

Primary example: `ai-bot.cn` synthetic user evaluation. The sample includes a newcomer, designer, product manager and privacy-cautious user. Reports show that users can identify search or category entry points, while also surfacing first-screen clarity, trust-signal visibility and path-efficiency issues.

Project link: https://github.com/Halucinaut/Doppel

Suggested tags: synthetic-user, experiential-testing, browser-agent, UX-evaluation.
